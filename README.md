# GWXtreme
Tools for inferences on extreme matter using gravitational wave parameter estimation data. For details on using this package please refer to the [documentation](https://gwxtreme.readthedocs.io/en/latest/).

Currently available tools:
--------------------------
1. Model selection of the neutron star equation of state using gravitational wave parameter estimation results. [1](https://doi.org/10.1103/PhysRevD.104.083003)
2. Inference over the parameters of parameterized equation of state models using MCMC sampling. The sampled posterior can then be translated into constraints on the pressure-density relation of cold nuclear matter. [2](https://doi.org/10.1103/PhysRevD.107.043035)
